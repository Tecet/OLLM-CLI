# Kiro Prompts Index

This folder contains reusable Kiro prompts used during development and review.

## How to Use

1. Pick a prompt file that matches your task (plan, execute, review).
2. Copy its contents into a Kiro session or CLI prompt.
3. Fill in the placeholders and follow the steps.

## Categories

- **Planning:** `plan-feature.md`, `create-prd.md`, `quickstart.md`
- **Execution:** `execute.md`, `implement-fix.md`, `execution-report.md`
- **Review:** `code-review.md`, `code-review-fix.md`, `code-review-hackathon.md`, `system-review.md`, `rca.md`
- **Bootstrap:** `prime.md`

## Contents

- `plan-feature.md` — feature planning workflow
- `implement-fix.md` — implementation guidance for fixes
- `execute.md` — execution checklist
- `execution-report.md` — structured execution report
- `code-review.md` — technical code review template
- `code-review-hackathon.md` — hackathon scoring rubric
- `code-review-fix.md` — targeted review for fixes
- `rca.md` — root cause analysis template
- `system-review.md` — system-level review
- `quickstart.md` — rapid project bootstrap
- `create-prd.md` — product requirements doc template
- `prime.md` — project priming template
