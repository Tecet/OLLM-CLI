# Keyboard Shortcuts

This document lists all available keyboard shortcuts in the OLLM CLI.

## Global Navigation
These shortcuts work from anywhere in the application.

| Key Combination | Action | Description |
|----------------|--------|-------------|
| **Tab** | Next Pane | Cycle focus to the next UI container (e.g., Input -> Chat -> Nav) |
| **Shift + Tab** | Previous Pane | Cycle focus to the previous UI container |
| **Ctrl + Space** | Focus Input | Jump focus directly to the Chat Input box |
| **Ctrl + M** | Focus Navigation | Jump focus directly to the Top/Left Navigation Bar |
| **Ctrl + C** | Focus Context | Jump focus directly to the Active Context panel |
| **Ctrl + F** | Focus File Tree | Jump focus directly to the File Tree panel |
| **Ctrl + I** | Focus Functions | Jump focus directly to the Functions/Info panel |

## Chat Interaction
available when Chat Input or Chat History is focused.

| Key Combination | Action | Description |
|----------------|--------|-------------|
| **Enter** | Send Message | Submit the typed message (Text Mode) |
| **Ctrl + PageUp** | Scroll Up | Scroll the chat history up |
| **Ctrl + PageDown** | Scroll Down | Scroll the chat history down |
| **Alt + Up** | Scroll Up | Alternative scroll up (if Ctrl+PageUp is intercepted) |
| **Alt + Down** | Scroll Down | Alternative scroll down |
| **Escape** | Cancel | Cancel current generation or exit current mode |

## Navigation Bar
Available when the Navigation Bar is focused (use `Ctrl+M` to focus).

| Key Combination | Action | Description |
|----------------|--------|-------------|
| **Left Arrow** | Previous Tab | Switch to the tab on the left |
| **Right Arrow** | Next Tab | Switch to the tab on the right |
| **Ctrl + 1** | Search Tab | Switch to Search tab |
| **Ctrl + 2** | Files Tab | Switch to Files tab |
| **Ctrl + 3** | GitHub Tab | Switch to GitHub tab |
| **Ctrl + 4** | Tools Tab | Switch to Tools tab |
| **Ctrl + 5** | Docs Tab | Switch to Docs tab |
| **Ctrl + 6** | Settings Tab | Switch to Settings tab |

## Menu Mode
Available when a menu is active (e.g., after a tool call or prompt selection).

| Key Combination | Action | Description |
|----------------|--------|-------------|
| **Up / Down** | Navigate | Move selection up or down |
| **Enter** | Select | Execute the selected option |
| **0** | Exit | Exit the menu |
| **9** | Back | Go back (if supported) |
| **1-8** | Quick Select | Directly select option 1-8 |
| **Escape** | Close | Close the menu without selecting |
